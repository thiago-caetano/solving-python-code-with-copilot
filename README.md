# Python com GitHub Copilot

Este repositório contém 6 exercícios básicos em Python, desenvolvidos com o apoio do GitHub Copilot.

## Questões resolvidas

### 1. Concatenando dados
Neste exercício, o programa recebe dois dados digitados pelo usuário e os junta em uma única frase.

### 2. Repetindo textos
Neste exercício, o usuário digita uma palavra ou texto e também um número inteiro. Depois, o programa repete o texto a quantidade de vezes informada.

### 3. Operação matemática simples
Neste exercício, o programa recebe dois números e realiza uma operação matemática simples entre eles, como por exemplo uma soma.

### 4. Verificando números pares e ímpares
Neste exercício, o programa recebe um número inteiro e verifica se ele é par ou ímpar utilizando estruturas condicionais.

### 5. Calculando a média de notas
Neste exercício, o programa recebe três notas e calcula a média usando operadores aritméticos.

### 6. Verificando palíndromos
Neste exercício, o programa recebe uma palavra e verifica se ela é um palíndromo, comparando a palavra original com sua versão invertida.

## Objetivo
Praticar lógica de programação em Python e usar o GitHub Copilot como apoio na criação dos códigos.

# Python with GitHub Copilot

This repository contains 6 basic Python exercises developed with the support of GitHub Copilot.

## Completed exercises

### 1. Concatenating data
In this exercise, the program receives two pieces of data entered by the user and joins them into a single sentence.

### 2. Repeating text
In this exercise, the user enters a word or text and an integer number. Then, the program repeats the text the number of times entered.

### 3. Simple mathematical operation
In this exercise, the program receives two numbers and performs a simple mathematical operation, such as addition.

### 4. Checking even and odd numbers
In this exercise, the program receives an integer and checks whether it is even or odd using conditional statements.

### 5. Calculating the average of grades
In this exercise, the program receives three grades and calculates the average using arithmetic operators.

### 6. Checking palindromes
In this exercise, the program receives a word and checks whether it is a palindrome by comparing the original word with its reversed version.

## Goal
Practice Python programming logic and use GitHub Copilot as support for writing code.
